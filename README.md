**RefScore AI**

**Academic Reference Quality Scoring System Using Machine Learning**

**Overview**

RefScore AI is a student project that uses machine learning to estimate the quality of academic references based on publication metadata such as year, citation count, publisher, number of authors, and document type. The goal is to support researchers and students in selecting more reliable and relevant sources for academic writing.
